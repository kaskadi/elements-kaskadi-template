# Setup before first commit

Before pushing for the first time, please setup secrets on this repository.

What needs to be done:
- `AWS_ACCESS_KEY_ID`
- `AWS_ACCESS_KEY_SECRET`

Those are the credentials of `kaskadi-public-push`
